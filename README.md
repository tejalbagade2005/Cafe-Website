🍽️ Spice Delight – Restaurant Website

Spice Delight ek modern Restaurant Website hai jo React.js se banayi gayi hai.
Is website par users restaurant ke menu, dishes aur contact information dekh sakte hain.

🚀 Features

. 🏠 Home Page with restaurant introduction

. 🍔 Food images and highlights

. 📖 About section

. 📞 Contact form

. 📱 Mobile friendly design

. 🎨 Clean UI using CSS


🛠️ Technologies Used

. React.js

. JavaScript

. HTML5

. CSS3
